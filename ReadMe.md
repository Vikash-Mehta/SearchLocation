##### Search Location
*Ruby on rails + React +  Webpack + React-leaflet + Postrgres
*
###### **Getting Started**

###### Frontend
- npm -v 6.4.1
- node -v 10.15.3
- tsc -v 3.5.2
- npm i
- npm start

###### Backend
- Install ruby, rails, gem
- rails -v 5.2.3
- gem -v 2.6.11
- bundle install
- rails s

###### DatabaseDatabase
- Import data from flyhomes.sql
- CREATE EXTENSION btree_gist;
